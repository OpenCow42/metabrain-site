# Agent Guidelines

These instructions apply to the entire `metabrain-site` repository.

## Release Checks

- Before releasing or publishing the website, check link validity.
- Validate internal anchors, local assets, and external URLs.
- If an external URL blocks command-line checks, verify it in a browser and
  document that result in the release notes or commit message.
- Do not knowingly ship broken source, package, install, image, or navigation
  links.
